Customizing.

Some template settings that will easily help you customize it.

How to insert or replace a photo / image:

By default, the template set image placeholder. These layers have the name of the image (smart-object). Double click on the layer image, insert your picture in the open document, and click Save Ctrl + S. All is ready!
